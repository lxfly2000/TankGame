# TankGame
这是我在学习编程时做的人生中第一个游戏程序，由于当时（2015年夏）用的库比较老，现在更新了一下放在Github上。

## 依赖项
* fmod 64位（已内置）
* [OpenCV 4.0](https://opencv.org)

## 运行条件
以前是支持32位系统的，但由于OpenCV4.0已不再提供32位预编译版本，所以现在只能运行在64位系统上。

## 控制方法
操作说明：
* Esc: 退出
* R: 重开
* T: 返回标题
* O: 自爆
* P: 暂停
* Space: 继续
* Z: 射击
* U/J/H/K: 移动

因为当时的编程能力有限，代码写得很渣，控制按键并没有使用方向键，如果你愿意可以自行修改代码。
